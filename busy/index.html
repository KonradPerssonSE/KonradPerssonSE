Rules (short form)
  • No backend. Portable static files.
  • Vanilla JS/CSS/HTML. Chrome latest only. No polyfills or prefixes.
  • Common files are optional + append-only: busy-common.css, busy-common.js.
  • Pages are self-contained toys. Use localStorage for state.
