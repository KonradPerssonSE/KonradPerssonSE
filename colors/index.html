Konrad Persson — Color System (OKLCH)

Warm parchment neutrals, deep teal-blue primaries, and metallic ochre/bronze accents. Built in OKLCH with relative color syntax so lightness shifts keep the same warm hue (no chalking).

Neutrals

All surfaces derive from --bg with oklch(from …) so the warm hue is preserved as you darken/lighten.

Semantics

Info. Use accent blue for border/icon; keep body text as ink.
Success. Deep moss gives AA contrast on parchment.
Warning. Ochre background with a clear border—no glare.
Danger. Amaranth background with strong border.

Tint backgrounds with oklch(from var(--color) …) (lift L, reduce C, keep H).

OKLCH “mixins”

Pattern used across this file—lighten/darken any token without shifting hue:

/* Set a working color */
.component { --_c: var(--primary); }

/* Then use tone steps */
background: var(--t300);   /* lighter tint  */
border-color: var(--t700); /* darker edge   */

Swap --_c for any token (e.g., --ochre) to get a consistent warm scale.
