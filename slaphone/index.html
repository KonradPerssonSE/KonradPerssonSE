Admin / intern instruktion (anställda)

Detta är en klient-side “toggle”. Det är inte ett säkerhetssystem. Byt PIN i koden innan du använder sidan skarpt.

Checklista: SLAPHONE Clean + Dumb Setup (500 SEK)

  1. Intag: notera modell, lagring, iOS-version, skärm/batteristatus, SIM-typ, kundens mål (SMS/samtal/kamera/offline).
  2. Apple ID / Find My: be kunden logga ut / stäng av “Hitta” vid behov. Målet: enheten ska vara enkel att återställa senare.
  3. Radera: Fabriksåterställ (Settings → General → Transfer or Reset → Erase). Sätt upp som Ny iPhone (inte backup).
  4. Uppdatera: uppdatera till senaste iOS som stöds för modellen (om möjligt) på Wi-Fi.
  5. Dumb-inställningar (minimera lagg & bakgrund):
    • Stäng av Background App Refresh, Location Services (om ej behövs), Auto-Updates, Analytics, Siri suggestions.
    • Stäng av iMessage/FaceTime om kunden vill ha “ren SMS/telefoni”.
    • Aktivera Reduce Motion / Reduce Transparency om kunden vill.
    • Aktivera Screen Time → blockera App Store-installation (valfritt, men rekommenderat).
  6. Offline-appar: installera bara det som beställts (t.ex. Books + VLC). Ladda test-PDF/MP3 om kunden vill.
  7. Test:
    • Samtal in/ut, SMS in/ut, kamera tar bild, Photos öppnar, lagring ok.
    • Om SIM ska flyttas: verifiera operatör/täckning och VoLTE-läge om tillgängligt.
  8. Hårdvara-tillägg:
    • SD/microSD: formatera exFAT, snabbtest läs/skriv på dator. Verifiera med kundens adapter.
    • M.2 NVMe: montera SSD i enclosure, formatera exFAT (single partition). Test på Windows + Mac/Linux om möjligt.
    • Batteripack: kontrollera laddning, kabeltyp, och att det kan ladda iPhone stabilt.
  9. Leverans: visa kunden “så här kopierar du foton” (kundens val: SD-flöde eller dator-import). Dokumentera i ordertexten.

Noteringar / kända constraints

  • Lightning på iPhone 8 är praktiskt USB-2 → ingen NVMe-hastighet vid telefonkopiering, men SSD är snabb på dator.
  • Externa lagringsenheter kan kräva extra ström via adapter/hubb (särskilt SSD).
  • SD-flöde är oftast enklast för kamera-folk. NVMe-flöde är bäst som “one portable work disk”.
